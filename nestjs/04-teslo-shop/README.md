# TesloDB API 

1. Levantar la base de datos.
``` 
docker-composer up -d 
```
2. Clonar el archivo ```.env.template``` y re nombrarlo a. ```.env```

3. Cambiar las varirables de entorno 

4. Levantar: yarn start:dev
